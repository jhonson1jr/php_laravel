# php_laravel
Projeto simples de PHP com Laravel

# Contem
 - Exemplos comentados de: Models, Migrations, Middlewares, Seeders dentre outros

 Os códigos estão em grande parte comentados e obedecem a estruturação que o Laravel opera.
